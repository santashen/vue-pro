欢迎使用个人信息管理系统

系统介绍

这是一个基于 Flask + Bootstrap 的个人信息管理系统，提供了完整的数据管理功能。 系统采用现代化的技术栈构建，具有以下特点：

技术栈
  • 🐍 Flask - Python Web 框架
  • 🗃️ SQLAlchemy - ORM 数据库操作
  • 🔄 Flask-Migrate - 数据库版本管理
  • 🌐 Flask-RESTful - RESTful API
  • 🎨 Bootstrap 5 - 现代化 UI 框架
主要功能
  • ✅ 个人信息的增删查改
  • ✅ 响应式卡片展示
  • ✅ RESTful API 接口
  • ✅ 数据库迁移管理
  • ✅ 现代化用户界面
快速开始

点击左侧导航栏的"数据管理"开始使用系统功能。

进入数据管理
系统状态
数据库状态: 正常
API 状态: 运行中
使用说明
📝 添加数据

在数据管理页面点击"添加新信息"按钮，填写表单即可添加新的个人信息。

✏️ 编辑数据

点击数据卡片上的"编辑"按钮，可以修改现有的个人信息。

🗑️ 删除数据

点击数据卡片上的"删除"按钮，确认后即可删除对应的信息。
